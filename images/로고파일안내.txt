로고 이미지 파일 안내
===================

제공받은 N CREATION 로고 이미지를 아래 파일명으로 이 폴더에 저장해주세요:

파일명: logo.png

권장 사양:
- 파일 형식: PNG (투명 배경)
- 권장 크기: 가로 400px ~ 800px (비율 유지)
- 배경: 투명 (white 로고)

현재 웹사이트에서는 다음과 같이 로고가 사용됩니다:
1. 헤더 (높이 35px) - 스크롤 전: 흰색, 스크롤 후: 검정색
2. 모바일 메뉴 (높이 35px) - 흰색
3. 푸터 (높이 40px) - 흰색

CSS 필터를 사용하여 자동으로 색상이 변경됩니다.

제공받은 로고 이미지:
- N CREATION 로고 (검정색 텍스트, 왼쪽에 N 심볼)
- 흰색 배경 또는 투명 배경
